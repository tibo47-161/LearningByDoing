HTML5 Semantik Modern Web Development

Lerne die Grundlagen der HTML5-Semantik und erstelle zugängliche, strukturierte Webseiten mit modernen Standards.

HTML5 Struktur 
<!DOCTYPE html>
<html lang="de">
<head>
    <meta charset="UTF-8">
    <title>Semantische HTML5</title>
</head>
<body>
    <header>
        <nav>Navigation</nav>
    </header>
    <main>
        <article>
            <section>Inhalt</section>
        </article>
    </main>
    <footer>Footer</footer>
</body>
</html>

Semantische HTML5 Elemente

Verstehe die Bedeutung und Verwendung der wichtigsten semantischen HTML5-Elemente.

<header>

Container

Definiert den Kopfbereich einer Seite oder eines Abschnitts.

Verwendung:

<header>
    <h1>Website Titel</h1>
    <nav>Navigation</nav>
</header>

<nav>

Navigation

Definiert einen Navigationsbereich mit Links.

Verwendung:

<nav>
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
    </ul>
</nav>

<main>

Hauptinhalt

Definiert den Hauptinhalt der Seite (nur einmal pro Seite).

Verwendung:

<main>
    <h1>Haupttitel</h1>
    <article>Artikel Inhalt</article>
</main>

<article>

Inhalt

Definiert einen eigenständigen, wiederverwendbaren Inhalt.

Verwendung:

<article>
    <header>
        <h2>Artikel Titel</h2>
    </header>
    <p>Artikel Inhalt...</p>
</article>

<section>

Abschnitt

Definiert einen thematischen Abschnitt innerhalb eines Dokuments.

Verwendung:

<section>
    <h2>Abschnitt Titel</h2>
    <p>Abschnitt Inhalt...</p>
</section>

<aside>

Nebeninhalt

Definiert Inhalte, die vom Hauptinhalt getrennt sind.

Verwendung:

<aside>
    <h3>Sidebar</h3>
    <ul>
        <li>Zusätzliche Links</li>
    </ul>
</aside>

<footer>

Fußbereich

Definiert den Fußbereich einer Seite oder eines Abschnitts.

Verwendung:

<footer>
    <p>© 2024 Website</p>
    <nav>Footer Navigation</nav>
</footer>

Accessibility (Barrierefreiheit)

Lerne, wie du zugängliche Webseiten mit HTML5 erstellst.

ARIA Labels

Verwende ARIA-Attribute für bessere Screen Reader Unterstützung.

<button aria-label="Close menu">
    <i class="fas fa-times"></i>
</button>

Semantische Struktur

Verwende die richtigen HTML5-Elemente für bessere Navigation.

<main role="main">
    <h1>Haupttitel</h1>
    <section aria-labelledby="section-title">
        <h2 id="section-title">Abschnitt</h2>
    </section>
</main>

Skip Links

Hilfe für Tastaturnutzer, direkt zum Hauptinhalt zu springen.

<a href="#main-content" class="skip-link">
    Skip to main content
</a>

HTML5 Media Elemente

Integriere Bilder, Videos und Audio mit modernen HTML5-Elementen.

Responsive Images

<picture>
    <source media="(min-width: 768px)" 
            srcset="large.jpg">
    <source media="(min-width: 480px)" 
            srcset="medium.jpg">
    <img src="small.jpg" alt="Beschreibung">
</picture>

HTML5 Video

<video controls preload="metadata">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <track kind="subtitles" src="subtitles.vtt">
    Fallback Text
</video>

HTML5 Audio

<audio controls preload="metadata">
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Fallback Text
</audio>
