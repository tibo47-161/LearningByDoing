Utility-First CSS Framework Tailwind CSS

Lerne Tailwind CSS von Grund auf - von einfachen Utilities bis hin zu komplexen Komponenten. Entdecke die Kraft des utility-first Ansatzes für moderne Webentwicklung.

Utility Classes

Tailwind CSS bietet hunderte von Utility-Klassen für Spacing, Farben, Typografie, Layout und mehr.

Spacing

Konsistente Abstände mit m-*, p-*, space-* Klassen

m-4 p-6
space-x-4 space-y-2
gap-6

Colors

Umfangreiche Farbpalette mit text-*, bg-*, border-* Klassen

text-blue-600
bg-gradient-to-r
border-red-500

Typography

Text-Styling mit text-*, font-*, leading-* Klassen

text-2xl font-bold
text-center leading-relaxed
uppercase tracking-wide

Layout

Flexbox und Grid Layout mit flex, grid, container Klassen

flex items-center
grid grid-cols-3
container mx-auto

Borders

Border-Styling mit border-*, rounded-* Klassen

border-2 border-gray-300
rounded-lg rounded-t-xl
shadow-lg

Effects

Transformationen und Effekte mit transform, filter Klassen

hover:scale-105
transition-all duration-300
backdrop-blur-sm

Dark Mode

Tailwind CSS unterstützt automatischen Dark Mode mit dem dark: Präfix.

Dark Mode Features

  • Automatische Erkennung der Systemeinstellung
  • Einfache Umschaltung mit JavaScript
  • Konsistente Farbpalette
  • Smooth Transitions

Code Example

// Dark mode classes
bg-white
dark:bg-gray-800
text-gray-900
dark:text-white
